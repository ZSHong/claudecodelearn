贪吃蛇游戏代码逻辑详解

一、HTML & CSS 结构

<canvas id="game" width="400" height="400"></canvas>

创建了一个 400×400 像素 的画布。

CSS 给画布加了 1px 黑色边框，并让它在页面中水平居中。

二、核心变量定义

const gridSize = 20;
const tileCount = canvas.width / gridSize; // 400 / 20 = 20
变量 含义
gridSize = 20 每个网格的边长是 20 像素
tileCount = 20 画布横向、纵向各有 20 个网格（400÷20） 
let snake = [{x: 10, y: 10}];  // 蛇初始在正中间，只有1节
let food = {x: 15, y: 15};     // 食物初始位置
let dx = 0, dy = 0;            // 蛇的移动方向（初始静止）
let score = 0;
坐标说明：游戏逻辑中使用的是网格坐标（0~19 的整数），而不是像素坐标。例如 {x:10, y:10} 代表第10列第10行。

三、游戏主循环 gameLoop()

这是整个游戏的心脏，每 100 毫秒 执行一次。

1. 初始静止状态

if (dx === 0 && dy === 0) {
    draw();
    setTimeout(gameLoop, 100);
    return;
}

如果玩家还没按方向键（dx=0, dy=0），只绘制画面，不移动蛇。

2. 计算蛇头新位置

const head = {x: snake[0].x + dx, y: snake[0].y + dy};

根据当前方向 dx, dy，计算蛇头下一步会到达哪个网格。

3. 穿墙逻辑

if (head.x < 0) head.x = tileCount - 1;   // 从左边出去，从右边进来
if (head.x >= tileCount) head.x = 0;       // 从右边出去，从左边进来
if (head.y < 0) head.y = tileCount - 1;   // 从上边出去，从下边进来
if (head.y >= tileCount) head.y = 0;       // 从下边出去，从上边进来

经典贪吃蛇穿墙设定：蛇头从一侧出去，会从另一侧进来。

4. 撞到自己检测

for (let segment of snake) {
    if (head.x === segment.x && head.y === segment.y) {
        resetGame();  // 撞到自己，游戏重置
        return;
    }
}

遍历蛇身，如果新蛇头和某节身体重合，判定为撞到自己，游戏结束并重置。

5. 移动蛇（核心逻辑）

snake.unshift(head);  // 在头部插入新位置

unshift() 把新位置加到数组最前面，新位置成为新蛇头。

6. 吃食物判断

if (head.x === food.x && head.y === food.y) {
    score++;
    food = {
        x: Math.floor(Math.random() * tileCount),
        y: Math.floor(Math.random() * tileCount)
    };
} else {
    snake.pop();  // 没吃到食物，就删掉尾部，保持长度不变
}
情况 行为
吃到食物 分数+1，重新随机生成食物位置，不删除尾部 → 蛇变长
没吃到食物 删除尾部 → 蛇向前移动一格，长度不变
关键设计：蛇的移动本质上就是「头部加一格，尾部删一格」；吃到食物时只加不删，长度就增加了。

四、绘制函数 draw()

// 背景
ctx.fillStyle = '#fff';
ctx.fillRect(0, 0, canvas.width, canvas.height);

// 蛇
ctx.fillStyle = '#0f0';
for (let segment of snake) {
    ctx.fillRect(segment.x * gridSize, segment.y * gridSize, gridSize - 2, gridSize - 2);
}

// 食物
ctx.fillStyle = '#f00';
ctx.fillRect(food.x * gridSize, food.y * gridSize, gridSize - 2, gridSize - 2);

// 分数
ctx.fillStyle = '#000';
ctx.font = '16px Arial';
ctx.fillText('分数：' + score, 10, 20);
  • 背景：白色填充整个画布
  • 蛇身：绿色方块，网格坐标转换成像素坐标（x * 20, y * 20），绘制大小是 18×18（gridSize - 2），留出 2 像素间隙
  • 食物：红色方块
  • 分数：左上角显示当前分数

五、重置函数 resetGame()

snake = [{x: 10, y: 10}];
dx = 0; dy = 0;
score = 0;
food = {x: 15, y: 15};

把所有状态恢复到初始值，蛇回到中心，方向归零，分数清零。

六、键盘控制

document.addEventListener('keydown', (e) => {
    switch(e.key) {
        case 'ArrowUp':    if (dy !== 1)  { dx = 0; dy = -1; } break;
        case 'ArrowDown':  if (dy !== -1) { dx = 0; dy = 1; }  break;
        case 'ArrowLeft':  if (dx !== 1)  { dx = -1; dy = 0; } break;
        case 'ArrowRight': if (dx !== -1) { dx = 1; dy = 0; }  break;
    }
});
按键 dx dy 方向
↑ 上 0 -1 向上
↓ 下 0 1 向下
← 左 -1 0 向左
→ 右 1 0 向右
防反向逻辑：例如 if (dy !== 1)，如果蛇正在向下走（dy=1），就不能按上键直接掉头，否则会瞬间撞到自己的身体。

七、总结：游戏流程

开始 → gameLoop
       ↓
    计算新蛇头(head)
       ↓
    穿墙处理
       ↓
    是否撞到自己？ ──是──→ resetGame()
       ↓ 否
    snake.unshift(head)
       ↓
    是否吃到食物？
     ↓ 是              ↓ 否
   score++          snake.pop()
   生成新食物          ↓
       ↓           draw()
    draw()         setTimeout(...)
       ↓              ↓
      等待100ms ←─────┘
